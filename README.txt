Сервис to-do-list для ведения заметок или отчетов о своих делах.
Используется Java 17 Corretto, Spring Boot 3.2.0 и SpringDoc OpenApi 2.3.0
Первым делом нужно создать пользователя у которого 2 поля: имя пользователя и его
почтовый адрес.
После этого уже можно будет создовать Таски у которого также 2 поля: описание таска
и его состояние: выполнено(true) и не выполнено(false).
Cервис имеет все нужные методы для получения,создания,удаления и обновления как юзеров,
так и тасков у юзера.
В сервис добавлена зависимость SpringDoc OpenAPI для поддержки интерактивной документации
Swagger. Htpp адрес документации следующее:
http://localhost:8071/toDoList-service/swagger-ui/index.html
Конфигурация базы данных находится в application.properties
В дальнешних раработках нужно будет создать файл docker-compose для контейнеризации
проекта
